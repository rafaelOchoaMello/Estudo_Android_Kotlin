64.Argumentos Variáveis

- Em suma, é a capacidade que se tem de passar uma quantidade mutável de parâmetros para um método.

- A assinatura base é 'vararg nomeDoParametro: tipoDoParametro'

Exemplo :: Método de ordenação de inteiros

	fun ordenar(vararg numeros: Int): IntArray {
		return numeros.sortedArray()
	}

	//Em algum lugar do código a chamada do método ficaria
	val valoresOrdenados = ordenar(38,3,10,1,22,13)

	//O restulado seria 1,3,10,13,22,38


- É importante notar, que é possível utilizar parâmetros 'comuns' ao lado dos 
variáveis, mas é precisa tomar um cuidado importante, a nomeação de especificação
do parâmetro 'comum'.

- A nomeação do parâmetro ocorre para dizer para o compilador que aquele não é 
outro parâmetro do vararg.


Exemplo :: Método de ordenação com vararg e outro parâmetro 'comum'

	fun ordenar(vararg numeros: Int, idDoUsuario: Int): IntArray {
		return numeros.sortedArray()
	}

	//Como podemos ver, ambos parâmetros são 'Int', desta forma será 
	preciso nomear o mesmo na chamada do método.

	val valoresOrdenados = ordenar(38,3,10,1,22,13, idDoUsuario = 1)
